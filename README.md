# Shark-Tank-India-Data-Analysis
This project analyzes Shark Tank India startup pitches, investments, industries, and sharks using python

## Tools Used
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Google colab

## Analysis Performed 
- Industry-wise startups
- Total investment analysis
- Shrak-wise investments
- Highest funded startups
- Deal status analysis

## Dataset
Shark Tank India Dataset
